This plugin connects your WooCommerce store to Cryptum Checkout Payment Gateway so you can start accepting cryptocurrencies.

Please notice that WooCommerce must be installed and active.


== Minimum Requirements ==

PHP 7.0 or greater is recommended
WordPress 5.7 or greater is recommended
WooCommerce 6.0 or greater is recommended

== Installation ==

To do an automatic install of the Cryptum Checkout, log in to your WordPress dashboard, navigate to the Plugins menu, and click “Add New.”

In the search field type “Cryptum NFT,” then click “Search Plugins.” Once you’ve found us, you can view details about it such as the point release, rating, and description. Most importantly of course, you can install it by Clicking “Install Now,” and WordPress will take it from there. Make sure that in addition to being installed, the plugin is active.

Go to the WooCommerce Section, click Settings, Press the “Payments” Tab at the top, Enable Cryptum Checkout, and press Manage.

Go to the Cryptum Dashboard and create your free account.

Click the ‘+ Project’ button and create a Store Project to receive your credentials.

Copy and paste your credentials - Store ID and API Key - into your WooCommerce settings.

Click the Save button and your store is connected to Cryptum dashboard.

Enter the wallets that will receive payments for each blockchain protocol in the Cryptum Dashboard.

Check more information in the complete <a href="https://doc.cryptum.io/main/for-sellers/overview-for-sellers" target="_blank">Documentation</a>.

== Development ==

This plugin is being developed on <a href="https://github.com/cryptum-official/cryptum-checkout-wordpress-plugin" target="_blank">GitHub</a> by Blockforce team. Feel free to look the code and help us.
The plugin connects with our Blockchain Cryptum Checkout APIs to register orders of cryptocurrency transactions. Those APIs will link the user’s payment with cryptocurrency and the order in your WooCommerce store.

For more details about Cryptum product go to our site.

== FAQ ==

= How much does it cost to use the plugins? =

To install and start using the plugins, you pay nothing!
You will need to set up a credit card so that Cryptum will charge a sales fee on each crypto payment received (Checkout) and on each completed sale.

= Do I need to develop something in code to be able to use it? =

No, our plugins are completely no-code! Anyone can use it, as long as they follow our step-by-step documentation.
If you are a developer, you can choose to use our Checkout API, available in our documentation.

= Which cryptocurrencies are accepted as payment? =

* Ether (ETH)

* Tether (USDT)

* USDC

* Celo (CELO)

* Celo Dollar (cUSD)

* Celo Real (cREAL)

* Hathor (HTR)
